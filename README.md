# 🐍 Logisim Snake Game

A digital logic implementation of the classic **Snake Game** built using **Logisim Evolution**. This project demonstrates how game mechanics can be designed using digital circuits, counters, registers, control logic, and finite state machine (FSM) concepts without traditional programming languages.

## 📖 Overview

The Snake Game is a popular arcade-style game where the player controls a snake that grows in length as it collects food. The objective is to achieve the highest possible score without colliding with the walls or the snake's own body.

This project recreates the game entirely using digital logic design principles in Logisim Evolution.

## ✨ Features

* 🎮 Fully functional Snake gameplay
* ⌨️ Directional controls (Up, Down, Left, Right)
* 🍎 Food generation and collection
* 📈 Score tracking system
* 🐍 Snake growth mechanism
* 💀 Collision detection
* 🔄 Game reset functionality
* 🛠️ Built entirely using Logisim Evolution

## 🚀 How to Run

### Requirements

* Logisim Evolution

### Steps

1. Download or clone this repository.
2. Open `SnakeGame.circ` using Logisim Evolution.
3. Start the simulation.
4. Use the control buttons/inputs to move the snake.
5. Collect food and avoid collisions.

## ⚙️ Design Concepts Used

This project applies several important Digital Logic Design concepts:

* Finite State Machines (FSM)
* Counters
* Registers
* Multiplexers
* Sequential Logic
* Combinational Logic
* Clock-based Synchronization
